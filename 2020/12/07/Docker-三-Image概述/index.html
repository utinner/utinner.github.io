Docker(三)Image概述

2020-12-07 运维 Docker 0 0 字数统计: 240(字) 阅读时长: 1(分)

什么是Image

  • 文件和meta data的集合（root filesystem）
  • 分层的，并且每一层都可以添加改变删除文件，成为一个新的image
  • 不同的image可以共享相同的layer
  • Image本身是read-only的

Image的获取

  • 通过Dockerfile构建
  • 从docker Registry获取
